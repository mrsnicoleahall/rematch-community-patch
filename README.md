# ➡️ This project has moved to **Long Live Pets**

**This repository is retired.** The old community Rematch patch that used to live
here has been replaced by a brand-new, **original** addon written from scratch:

## 👉 https://github.com/mrsnicoleahall/LongLivePets

**Long Live Pets** is an independent, clean-room pet team manager + collection
browser for World of Warcraft: *Midnight* (Interface 12.x). It does everything
the old patch did and a lot more — one window, drag-and-drop, move selection,
groups, notes, win/loss, send-to-player, and an **auto counter-team builder**
that learns tamer comps from your battles. It contains **no Rematch code**.

- **Get it / download:** https://github.com/mrsnicoleahall/LongLivePets/releases
- **Source & docs:** https://github.com/mrsnicoleahall/LongLivePets

### Why the move?
The old patch was built on top of **Rematch** (by Gello), which is not open
source. To keep things clean and legal, Long Live Pets is a fresh, original
implementation under the **MIT License** — free to use, read, and fork.
